# Truss Bridge Tension & Compression Simulator

TypeScriptとHTML5 Canvasで実装された、リアルタイムのトラス橋（構造力学）物理シミュレーターです。
構造材にかかる「張力（引っ張る力）」と「圧縮力（押しつぶす力）」をリアルタイムに計算し、色で視覚化します。

## 🚀 特徴

- **力学の視覚化（カラーマッピング）**:
  - <span style="color: red;">■</span> **赤色**: 張力（Tension / 引っ張り）
  - <span style="color: blue;">■</span> **青色**: 圧縮力（Compression / 押し潰し）
  - <span style="color: green;">■</span> **緑色**: 安定（無負荷）
- **インタラクティブな操作**: 橋の接合部（ノード）をマウスでドラッグして、任意の方向に強制的な負荷（荷重テスト）をかけることができます。
- **軽量・高速**: 物理演算には位置ベース物理（ベレ積分と距離拘束の反復解決）を採用しており、ブラウザ上で軽量かつ安定して動作します。

## 🛠️ 技術スタック

- **言語**: TypeScript
- **描画**: HTML5 Canvas API
- **開発ツール/ビルド**: Vite
- **物理アルゴリズム**: Verlet Integration (ベレ積分) + Distance Constraints

📐 物理・力学的な背景
このシミュレーターは、実際の構造力学における「ワーレントラス（Warren Truss）」構造をベースにしています。
橋が自重や荷重によって下方にたわむ際、力学の法則通りに以下の現象が視覚的に確認できます。

上弦材（天井の梁）: 押しつぶされるため、青色（圧縮）に変化します。

下弦材（床面の梁）: 左右に引き伸ばされるため、赤色（張力）に変化します。

斜材（ジグザグの梁）: トラスの三角形構造により、一本おきに赤と青の負荷が交互に分散されます。

💡 今後の拡張アイデア（カスタマイズ例）
構造材の破断（破壊演出）: 一定以上の負荷（stress）がかかった際にリンクを削除する処理を追加すれば、重荷で崩壊する橋のシミュレーションが可能です。

カスタムビルドモード: ユーザーが自由にノードを配置してオリジナルの橋を設計できるゲーム要素の追加。
